ALT (alternative/alternativo) - É o elemento que será apresentado caso a imagem não carregue.

O atributo alt é MUITO IMPORTANTE

  • Acesseibilidade
  • Leitores de tela
  • SEO
  • Fallback

Ajustando o tamanho das imagens

HEIGHT - O atributo height serve para ajustar o tamnho da altura da imagem.

WIDTH - O atributo width serve para ajustar o tamanho da largura da imagem.

Trabalhando com imagens responsivas

O picture é um elemento do HTML usado para imagens responsivas e adptativas.

Essa tag, permite o seguinte:

  • Tricar a imagem conforme o tamanho da tela
  • Carregar imagns diferentes para mobile, tablet e desktop.

srcset - serve para oferecer multiplas versões de uma mídia e deixar o navegador escolher a mais adequada conforme tamanho da tela, formato suportado, largura disponível, performance.
