# Introdução ao HTML

Este repositório fala dos fundamentos do **HTML (Hypertext Narkup Languege)** 
e os principais elementos utilizados para o desenvolvimento de websites

## Principais elementos
Abaixo, temos a lista dos principais elementos visto em sala de aula:

- Titulos: **(h1...h6)**
- Parágrafos: `<p></p>`
- Listas:
    - OL: Lista ordenada
    - UL: lista nao ordenada
- Imagens: `<img>`
- links: `<a></a>`
- Forms: `<form></form>`
