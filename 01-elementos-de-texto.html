HTML

Introdução

HTML é a linguagem responsável pela estrutura das páginas web. Ela foi criada por Tim Berners-Lee e é utilizada em praticamente todos os sites da internet.

Importância do HTML

O HTML é extremamente importante para o desenvolvimento web. Muitos desenvolvedores realmente começam seus estudos aprendendo HTML antes de CSS e JavaScript.

Estrutura de uma página

Uma página HTML normalmente possui elementos como html, head e body.

Elemento body

O elemento body representa o conteúdo visível da página.

Elemento strong

A tag possui valor semântico e indica importância do conteúdo.

Elemento b

A tag b serve para aplicar negrito visual

Elemento em

O elemento em serve para dar ênfase a palavras ou frases.

Elemento i

O elemento i aplica apenas um efeito visual em italico
