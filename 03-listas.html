Entendendo as listas do HTML

As listas no HTML, servem para organizar informações relacionadas de forma estruturada e semantica.

As listas são compostas por elementos chamados de LI.

Os LI (List Item da lista). As listas não possuem limite de itens.

Ordered List - OL

O elemento OL serve para criarmos uma lista ordenada, ou seja, com ordem, importância.

Lista de compras

  1. Banana
  2. Maça
  3. Melão
  4. Uva

Unordered List

Serve para criar listas sem ordem específica. Normalmente aparece com bolinhas (bullets)

Exemplos rápidos

  • Trabalhar
  • Visitar minha tia
  • Ir ao parque
