Conhecendo o elemento de navegação

Elemento a

Um link é um elemento que permite navegar de uma página, documento, ou recurso para outro.

No HTML, a gente usa os links para:

  • Abrir um site externo
  • Levar o usuário para outra página do site
  • Iniciar dowload de arquivos
  • Abrir aplicativos como e-mail e telefone

Conhecendo elemento a

O elemento usado para criar links é a tag a (anchor)

href - hyperText Refrence

target - define onde o conteúdo do link será aberto quando o usuário clicar nele.

  • _self: Valor do navegador. Indica que o site será aberto mesma aba.
  • _blank: Abre o link em uma nova aba ou janela dependendo dp navegador.
  • _parents: Usamos quando a página está dentro de um frame/iframe, ou seja, o link abre o fram no pai.
  • _top: Abre o link ocupando toda a janela do navegador, saindo d etodos os frames.
